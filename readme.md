# 👥 KeenKeeper — Keep Your Friendships Alive

**KeenKeeper** is a purposeful relationship management tool designed to help users track and nurture meaningful connections. This application ensures you never lose touch with those who matter most by monitoring contact frequency and providing visual status indicators.

## 🚀 Key Features
* **Dynamic Dashboard Summary:** A top-level view of social health (Total Friends, On-Track, Overdue).
* **Friendship Status Tracking:** Color-coded system (On-Track, Almost Due, Overdue) matching Figma specs.
* **Interactive Timeline (C2):** Log and filter interactions by type (Call/Text/Video).
* **Friendship Analytics (C1):** Recharts Pie Chart visualizing interaction distribution.
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop.

## 🛠️ Technologies Used
* **Next.js / React:** Core framework and App Router.
* **Tailwind CSS & DaisyUI:** Pixel-perfect styling.
* **Recharts:** Data visualization.
* **Context API:** Global state management for friendship logs.
* **React Toastify:** Instant user feedback.
